Make good use of

Helper Classes

A list of helpful classes
  • .section-size-[1-12/full]

    Used to define the size of your section.

  • .font-size-[1-12]

    Used to define font-sizes.

  • .pad-[1-12]

    Add padding to a DOM element.

  • .space-[1-12]

    Add space (margin) to a DOM element.

  • .n-h-space-[1-12]

    Define negative horizontal space. Try it!

  • .n-v-space-[1-12]

    Define negative vertical space.

  • .width-[1-12]

    Define a custom width

  • .opacity-[1-10]

    Make an element less opaque: more transparent.

  • .relative

    Make the element have a relative position.

  • .absolute

    Make the element have a absolute position.

  • .fixed

    Make the element have a fixed position.

  • .centered

    Centers an element horitonzally.

  • .center

    Centers an element both horizontally and vertically. Uses flexbox.

  • .text-left

    Left align text

  • .text-center

    Center align text

  • .text-right

    Right align text

  • .pull-left

    Pulls an item to the left

  • .pull-right

    Pulls an item to the right

  • .light

    A light background

  • .lighter

    A lighter background

  • .grey

    A grey background

  • .black

    A black background

  • .light-text

    Light text

  • .black-text

    Black text

  • .border-top

    Sets a border on top. Try also: -bottom, -left, -right.

  • .shadow

    Sets a neat shadow on the element. Cool!

  • .rotate45

    Rotates an element 45 degrees.

  • .rotate70

    Rotates an element 70 degrees.
