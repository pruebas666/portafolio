Nevo Lightbox

Setting up

So you'd like to use the lightbox? Simply copy the markup that exists within the portfolio-lightbox.html example.

Minimum requirements

To have a working lightbox, you'll need at least:

  • A wrapper (div) that contains the ".gallery" class
  • A child element (a link) where you set the href as the lightbox image.
The most simple example
Hey there, I'm a lightbox item

div.gallery
  a.lightbox(href = '/path/to/image.jpg')

As you can see, there's just one image that loops. Cool huh? Have you tried zooming it as well?
